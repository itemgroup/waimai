:cherries: 整个项目是为了加强自己对于鸿蒙开发的理解而完成的，主要的目的有几个：
* 加强鸿蒙中UI界面的绘制
* 学会在鸿蒙APP中访问远程服务器
* 加强鸿蒙线程管理

---


 **主要的技术栈** ：
 * :pig2: 后端使用SpringBoot、JWT、MyBatis
 * :monkey: 前端使用鸿蒙提供的组件
 * :ghost: 数据库使用Mysql
